Willkommen zu Ihrem Repository Programmieren in Python

Das Repository ist initialisiert mit einem Ordner für Ihre Python-Programme

    PyPrgs

in die Sie alle Ihre Aufgaben machen sollten.    

Auf der Konsole (oder halt mit klicken) checken Sie das Repository aus mit
    git clone https://scm.inftech.hs-mannheim.de/scm/git/20pythxx
wobei xx Ihre persönliche Nummer ist

Konfigurieren Sie sich Visual Studio Code wie in den Installationshinweisen
InstallPython.pdf beschrieben. Starten Sie Visual Studio Code und öffnen Sie
den Ordner PyPrgs.
Alle Ihre Python-Programme sind Module, also Dateinamen mit .py am Ende.
Der Dateinamen besteht nur aus Kleinbuchstaben, keine Ziffern am Anfang,
keine Leer-/Sonderzeichen. Module sind Dateien, nicht Ordner. Wir nehmen
keine Unterordner! (nehmen Sie Ordner wird es erst mal schief gehen; don't).
